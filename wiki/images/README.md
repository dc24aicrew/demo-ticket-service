# Wiki Images

This directory should contain images referenced in the wiki documentation.

The following images are needed:

1. `clean_architecture.png` - Clean Architecture diagram
2. `login_screen.png` - Screenshot of the login screen
3. `dashboard.png` - Screenshot of the main dashboard
4. `ticket_search.png` - Screenshot of the ticket search functionality
5. `ticket_details.png` - Screenshot of ticket details view
6. `update_status.png` - Screenshot of ticket status update
7. `events_list.png` - Screenshot of events list
8. `event_tickets.png` - Screenshot of event tickets

## Adding Images

When adding images to this directory, please:

1. Use descriptive filenames
2. Optimize images for web display (compress if needed)
3. Use PNG format for screenshots and diagrams
4. Update wiki pages to reference the new images

## Image Usage in Wiki

To reference images in wiki pages, use the following Markdown syntax:

```markdown
![Alt Text](https://raw.githubusercontent.com/wiki/dc24aicrew/demo-ticket-service/images/image_name.png)
```

Replace `image_name.png` with the actual image filename.
